# VibeNet Frontend

React.js client for **VibeNet** — a scalable, end-to-end encrypted (E2EE) real-time chat application.

This repository is consumed as a git submodule by the [VibeNet-Main](https://github.com/ChamathDilshanC) monorepo.

## Tech Stack
- React.js (Vite)
- TweetNaCl.js for client-side E2EE
- WebSocket client for real-time messaging
